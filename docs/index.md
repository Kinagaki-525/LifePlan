# ライフプランシミュレーター仕様書（統合版）

## 0. 位置づけ

本書は `docs/index.md` と `docs/web-app-spec.md` を統合した実装基準仕様（v1.1）とする。実装時は本書を正とし、未確定事項は `docs/specs/open-issues.md` で管理する。

## 1. 概要

`rennsyu` の ASP.NET Core MVC アプリに、家計の将来推移を可視化するライフプランシミュレーターを追加する。

- 対象：`Home/Index` の単一画面
- 利用前提：夫婦での利用。単身利用は対象外
- 単位：入力は万円、内部計算は円、表示は万円
- 出力：サマリー、年次キャッシュフロー表、貯蓄推移グラフ
- 期間：現在年から夫・妻の両方が100歳に到達する年まで
- 対象外：保存、共有URL、PDF出力、メール登録、課金

> 旧仕様は90歳までだったが、`docs/web-app-spec.md` の詳細表が100歳前提のため、本書では夫・妻それぞれ100歳を終端とする。

## 2. 画面構成

入力エリアは、家族構成、ライフイベント、貯蓄、収入、支出、シミュレーション実行ボタンの順で表示する。結果エリアは初期表示時にプレースホルダーを表示し、試算後にサマリー、年次キャッシュフロー表、貯蓄推移グラフの順で表示する。

## 3. 入力仕様

### 3.1 家族構成

- 夫年齢、妻年齢：必須、16〜100の整数。両方入力されていない場合は試算不可
- 子ども1〜4年齢：任意、-20〜30の整数または `-（なし）`
- 子どもの負数年齢は将来出生予定として扱う（例：-3 は3年後出生）

### 3.2 ライフイベント

- 結婚：費用、実施年齢（夫年齢基準）
- 住宅購入：頭金、借入額、購入時期、ローン年数、想定金利
  - 住宅購入費（諸費用含む）は `頭金 + 借入額`
  - 購入時期未入力時は、頭金・借入額・ローン年数・想定金利を入力不可
- 自動車：購入額、初回購入年齢、買い替え間隔
- 教育費：第1〜4子ごとに、保育園〜大学院の区分を選択
  - 在籍期間は、保育園（0〜2歳）、幼稚園・保育園（3〜5歳）、小学校（6〜11歳）、中学校（12〜14歳）、高校（15〜17歳）、大学（18〜21歳）、大学院（22〜23歳）
  - 留年・浪人は考慮しない
- 旅行・その他：年間費用、開始年齢、終了年齢

### 3.3 貯蓄・収入・支出

- 貯蓄：現在の金融資産残高、想定運用年利（0〜20%）
- 収入：夫・妻それぞれ、職業、年収（手取り）、年収変化、就労開始/終了年齢、退職金、年金年額、年金受取開始年齢
- 支出：毎月の基本生活費、想定インフレ率（0〜5%）、毎月の家賃、その他支出（年間）
- 収入はすべて手取り前提とし、税・社会保険料は計算しない
- 年金の参考値は、厚生年金＋国民年金、国民年金のみの男女別水準を基準とする

### 3.4 入力方式とバリデーション

- 年齢、期間、区分、職業など選択肢が決まっている項目はプルダウンを基本とする
- 金額、年収、退職金、年金、家賃、生活費などは自由入力を基本とする
- 利率は数値入力、単位は `%` とする
- 結婚年齢は過去・現在・未来のいずれも入力可。夫の現在年齢以上には制限しない
- 就労終了年齢は就労開始年齢以上、年金受取開始年齢は就労終了年齢より後とする
- 旅行・その他の終了年齢は開始年齢以上とする
- 自動車の初回購入年齢が未入力の場合、買い替え間隔は無効扱いとする
- 子ども年齢が `-（なし）` の場合、その子どもの教育費入力は無効扱いとする

## 4. 計算仕様

### 4.1 試算期間と年齢

- 開始年は実行時の現在年、終了年は夫・妻の両方が100歳に到達する年
- 年次単位で計算し、内部で必要な場合は月次按分を利用する
- 夫・妻・子どもの年齢は初年度入力値を基準に毎年+1する
- 夫・妻はそれぞれ100歳を終端とし、100歳到達翌年以降は空白にする
- 子どもの初年度が空なら当該行は非表示。負数年齢は表では非表示、内部判定では使用可

### 4.2 収入

- 給与：就労開始〜終了年齢に計上し、設定した年率で毎年増減
- 退職金：就労終了年齢年に単発計上
- 年金：受給開始年齢以降に毎年計上
- 収入合計：夫・妻の給与、退職金、年金を合算
- 給与・退職金・年金は手取り金額として扱う

### 4.3 支出

- 基本生活費：`月額 × 12` にインフレ率を年次適用
- 家賃：住宅購入年から0
- その他支出：年間固定額を計上
- 結婚：指定年齢に単発計上
- 自動車：初回購入年齢から買い替え間隔ごとに単発計上
- 教育費：学校区分マスタに応じて対象年齢で計上
- 旅行・その他：開始〜終了年齢で毎年計上
- 支出合計：上記項目を合算。同一年に複数支出が重なる場合もそのまま合算

#### 住宅ローン

- 購入年に頭金を一括計上する
- 借入額は直接支出にせず、借入額を元本にした年額ローン返済額を計上する
- 購入年は頭金と年額ローン返済額、次年度以降は年額ローン返済額のみを計上する
- 年額ローン返済額は、借入額を元本、想定金利を年利、ローン年数を返済年数とした元利均等返済で算出する
- 年利0%の場合は `借入額 ÷ ローン年数` を年額ローン返済額とする
- ローン年数は返済を計上する総年数とし、購入年を1年目として数える
- 例：購入年齢30歳、ローン年数35年の場合、30歳年から64歳年まで35年分を計上する

#### 教育費

初年度と次年度以降の金額が異なる区分は、各在籍期間の初年度のみ初年度金額を適用し、2年目以降は次年度以降金額を適用する。

### 4.4 貯蓄

- 初年度の開始金融資産は、現在の金融資産残高とする
- 収支差額：`収入合計 - 支出合計`
- 貯蓄合計（0%運用）：`前年末残高 + 当年収支差額`
- 貯蓄合計（想定年利運用）：`前年末残高に想定運用年利を適用 + 当年収支差額`
- 想定運用年利は、現在の金融資産残高を起点とする資産全体に適用する

### 4.5 端数処理

- 内部計算は円単位で行う
- 年額ローン返済額など、計算結果に小数が出る場合は円単位で四捨五入する
- 画面表示時は万円単位に換算する

## 6. 表示仕様

### 6.1 サマリー

- 試算開始年/終了年
- 最終年の貯蓄残高（0%運用/年利運用）
- 最低残高年と金額
- 大型支出（住宅・教育・車など）のピーク年

### 6.2 キャッシュフロー表

- 家族年齢：夫、妻、子ども
- 収入系：給与、退職金、年金、合計
- 支出系：生活費、家賃、イベント群、合計
- 貯蓄系：開始資産、収支差額、残高推移

### 6.3 グラフ

`wwwroot/js/life-plan-chart.js` で、収入合計、支出合計、貯蓄合計（0%運用/想定年利運用）を描画する。
