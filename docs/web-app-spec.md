# WEBアプリ1 仕様書

## 1. 概要

- 対象アプリ：ライフプランシミュレーター Webアプリ
- 目的：家族構成・収入・支出・ライフイベントを入力し、将来のキャッシュフローと貯蓄推移を確認できるようにする
- 利用前提：夫婦での利用を対象とし、単身利用は対象外とする
- 金額単位：万円
- 参照元メモ：`WEBアプリ1` Excelファイル
- 参考URL：https://www.promise-plaza.com/useful/
